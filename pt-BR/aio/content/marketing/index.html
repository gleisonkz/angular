Desenvolva em todas as plataformas

Aprenda uma maneira de construir aplicações com Angular e reuse seu código e habilidades para construir aplicações para qualquer plataforma. Para web, web móvel, aplicativos nativo e desktop nativo.

Velocidade e Desempenho

Alcance a velocidade máxima possível na Plataforma Web hoje, e leve adiante, via Web Workers e renderização no lado do servidor (server-side rendering).

O Angular coloca você no controle sobre escalabilidade. Atenda a enormes requisitos de dados construindo modelos de dados em RxJS, Immutable.js ou outro modelo push.

Ferramentas Incríveis

Crie recursos rapidamente com temas simples e declarativos. Estenda a linguagem de template com seus próprios componentes e use uma grande variedade de componentes existentes. Receba ajuda e feedback específicos do Angular para quase todas as IDE e editores. Tudo isso vem junto, então você pode focar em construir aplicações incríveis ao invés de tentar fazer o código funcionar.

Amado por milhões

Desde o protótipo através da implantação global, Angular oferece a produtividade e a infraestrutura escalável que suporta as maiores aplicações do Google.
