Funcionalidades e Benefícios

Multi Plataforma
Aplicações Web Progressivas

Usar recursos modernos de plataforma web para fornecer experiências como de aplicativos nativos. Instalação de alto desempenho, off-line e em zero passos.

Nativo

Construa aplicativos móveis nativos utilizando estratégias como o Cordova, Ionic ou NativeScript.

Desktop

Crie aplicações desktop para Mac, Windows, e Linux usando os mesmos conceitos do Angular que você aprendeu para a web mais a capacidade de acessar as APIs nativas do SO (Sistema Operacional).

Velocidade e Desempenho
Geração de código

Angular transforma seus templates em código altamente otimizado para máquinas virtuais de JavaScript de hoje, oferecendo a você todos os benefícios do código escrito à mão com a produtividade de um framework.

Universal

Sirva a primeira página de sua aplicação no Node.js, .NET, PHP e outros servidores para renderização instantânea em apenas HTML e CSS. Também abre caminho para os sites que otimizam o SEO.

Separação de código

Aplicações Angular carregam rapidamente com o novo Componente de Router, que proporciona separação automática de códigos, para que os usuários só carreguem o código necessário para processar a página solicitada.

Produtividade
Templates

Crie rapidamente interfaces de páginas da web utilizando uma sintaxe de template simples e poderosa.

Angular CLI

Ferramentas de linha de comando: comece a construir rapidamente, adicione componentes e testes, e então implante instantaneamente.

IDEs

Receba sugestões inteligente de código, erros instantâneos e outros comentários em editores e IDEs populares.

História completa de desenvolvimento
Testes

Com Karma para testes de unidade, você pode saber se você quebrou as coisas toda vez que salva algum arquivo.

Animações

Crie animações e movimentações complexas de alto desempenho com um código curto através da API intuitiva do Angular.

Acessibilidade

Crie aplicações acessíveis com componentes habilitados por ARIA, guias para desenvolvedores e infraestrutura de testes embutida para a11y.
